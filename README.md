### **General info**
- App is made for eductional purposes (CompanyApp-MongoDB, practising MongoDB) during Web Developer Bootcamp 2020.

### **Running the project**
- One can run this project with command:
```javascript
yarn start 
```
or
```javascript
npm start 
```
### **Running tests**
- One can run tests (Mocha, Chai) with command:
```javascript
yarn test 
```
or
```javascript
npm test 
```
